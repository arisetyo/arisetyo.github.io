Colophon

Technologies used to build this website:

  1. Three.js: 3D objects/animations, base for shaders, post-processing effects
  2. GLSL: shaders animation and effects
  3. Alpine.js: page interacvity
  4. GSAP: custom animations, transitions, and scroll-trigger
  5. Anime.js + Splt.js: text (menus) animations
  6. Sass: better CSS structure
  7. Lenis: smooth scrolling
  8. Vite: for development and build phase

Software used for design & development:

  1. Visual Studio Code: code editor
  2. Figma: design and prototyping
  3. Github Copilot: AI pair programming
  4. Github Pages: hosting
